SELENA - Simple Virtual Assistant
(moved repo https -> ssh)

SAV is a very simple application implementing tooltips as a virtual assistant.
Requires jQuery, jQuery cookie plugin and Twitter Bootstrap (all included).

5. changing Selena's response text:

<div class="sva-[class name]">hoover me!</div>

where: [class name] - corresponding to <id> element in .xml file with Selena's phrases:

-- sample.xml --
<items>
    <item>
	<id>sample01</id>
	<text>This is sample solution for Your problem number 1</text>
    </item>
    <item>
	<id>sample02</id>
	<text>This is my proposition for problem number 2 :)</text>
    </item>
</items>
